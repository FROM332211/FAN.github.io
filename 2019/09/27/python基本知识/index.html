python基本知识

发表于 2019-09-27

###Python中的一些基本知识
缩进：表达程序格式框架，表达代码间包含和层次关系，一般4空格或1tab。
注释：

变量：用来保存和表示数据的占位符号
变量采用标识符来表示，关联标识符的过程叫命名
命名规则：大小写字母，数字，下划线和汉字等字符及组合。（大小写敏感，首字符不能是数字，不与保留字相同）
数据类型：供计算机程序理解的数据形式

赋值语句：由赋值符号构成的一行代码，用来给变量赋予新的数据值。
分支语句：由判断条件决定程序运行方向的语句。（if,elif，else….）
函数：根据输入参数产生不同输出的功能过程。（函数名（参数）
标准库：随解释器直接安装到操作系统中的功能模块
第三方库：需要经过安装才能使用的功能模块
库Library，包Package，模块Module，统称模块
库引用：扩充Python程序功能的方式：
import<库名>
<库名>.<函数名>(<函数参数>)
from <库名> import <函数名>
<函数名>.(<函数参数>)
import<库名> as<库别名>
<库别名>.<函数名>(<函数参数>)
循环语句：按照一定次数循环执行的一组语句
for <变量> in range (<参数>) 从0开始直到参数的值-1
整数：与数学中整数概念一致，可正可负，没有取值范围
浮点数：带有小数点及小数的数字，可能会出现不确定位数的情况
科学计数法：(a)e(b) 表示a乘于10的b次方
复数：数学中的复数
